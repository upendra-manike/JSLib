# Changelog

All notable changes to Next Motion Kit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-11-13
### Added
- Initial release of Next Motion Kit
- MotionHero, MotionSection, MotionCounter, MotionNavbar, MotionPageTransition, MotionReveal components
- Hooks: useAnimateOnScroll, useScrollProgress
- Motion presets helper utilities
- TypeScript definitions and Framer Motion integration
- Documentation, examples, and AI-friendly api.json
