# Changelog

All notable changes to SEO Boost will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-11-13

### Added
- Initial release of SEO Boost
- Meta tag manager with support for title, description, canonical, keywords, and custom meta tags
- Open Graph meta tags support for social media sharing
- Twitter Cards support for Twitter sharing
- JSON-LD structured data support with helper functions for common schemas
- Sitemap generator for XML sitemaps
- Robots.txt generator
- SEO analyzer with score calculation and recommendations
- React hook `useSEO` for easy integration
- TypeScript support with full type definitions
- SSR and SPA support
- Comprehensive documentation and examples
- AI-friendly API documentation

### Features
- **Meta Tags**: Complete meta tag management
- **Open Graph**: Full Open Graph protocol support
- **Twitter Cards**: All Twitter Card types supported
- **JSON-LD**: Helper functions for Organization, WebSite, Article, Product, BreadcrumbList schemas
- **Sitemap**: XML sitemap generation with all standard fields
- **Robots.txt**: Dynamic robots.txt generation
- **SEO Analyzer**: Comprehensive SEO analysis with scoring
- **React Hook**: Easy-to-use hook for React applications
